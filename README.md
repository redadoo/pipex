## Usage

To run `pipex` follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/redadoo/pipex.git
```

2. enter pipex Folder
```bash
cd pipex/
```
3. compile
```bash
make 
```
4. test
```bash
 make exe
```
```bash
 make exe1
```
